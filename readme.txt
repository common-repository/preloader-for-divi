Divi Preloader adds preloader to your Divi Based site.

== Description ==

Divi preloader is the best plugin to add a preloader to your Divi based website.

### Divi Preloader Features
- Retina Ready (most of the preloaders are in SVG format so provide best quality results.)
- Custom background color.
- 15+ Amazing options to display Preloader.
- Minimal code, so it's fully optimized.
- Change preloader sizes very easily.
- Control the visibility on mobile and desktop.


== Installation ==

1. Upload the entire folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Divi Preloader settings from the WordPress dashboard menu.
4. Configure the settings for Divi Preloader to make it work.
